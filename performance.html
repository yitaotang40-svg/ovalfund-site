如何更新数据（最简单）

  1. 打开 data/performance.json，把示例数据替换成你的真实数据。
  2. 保持字段：date（YYYY-MM-DD）、fund、sp500、brk（都用 Base=100 的净值）。
  3. 提交到 GitHub，Pages 会自动更新。

如果你后续要自动更新：可以用脚本每天生成 JSON（或 CSV），再用 GitHub Actions 定时提交。
